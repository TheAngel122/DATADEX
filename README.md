# DATADEX
Enciclopedia Pokémon interactiva desarrollada con Vanilla JavaScript y CSS3. Consume la PokéAPI para mostrar datos en tiempo real de +1000 Pokemons. Incluye buscador avanzado, filtrado elemental, gráficas de estadísticas con Chart.js, cálculo dinámico de debilidades, línea evolutiva y efectos visuales de alta fidelidad.
